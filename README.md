# Cam Shooter

A browser-based shooting game that uses your webcam and hand tracking to aim and shoot targets.

## Features

- Hand tracking using webcam for aiming
- Interactive shooting gameplay
- Real-time target tracking
- Score tracking and game HUD

## Technologies

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- MediaPipe/Hand tracking

## Setup

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## How to Play

1. Allow webcam access when prompted
2. Use your hand movements to aim the crosshair
3. Shoot targets to score points
4. Track your performance on the HUD
